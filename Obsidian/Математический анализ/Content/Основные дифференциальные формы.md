# Для скалярных полей 
$\omega^{0}_{f}(x)=f(x)-$ [[Дифференциальная форма|дифференциальная форма]] степени 0
$\omega^{n}_{f} (x)= f(x)(dx_{1}\wedge..\wedge dx_{n})-$ [[Дифференциальная форма|дифференциальная форма]] объёма
